# Wiki Content for GitHub

This directory contains documentation formatted for the GitHub Wiki.

## Publishing to Wiki

To add these pages to your GitHub Wiki:

1. Go to https://github.com/Gad9889/STM32-Platform/wiki
2. Click "Create the first page" or "New Page"
3. Copy content from the `.md` files in this directory
4. Create pages with these names:
   - Home
   - Getting-Started
   - API-Reference
   - Architecture
   - Examples
   - Migration-Guide
   - VS-Code-Extension

## Pages in this Directory

- `Home.md` - Wiki homepage with overview
- `Getting-Started.md` - Installation and quick start
- `API-Reference.md` - Complete API documentation
- `Architecture.md` - System design
- `Examples.md` - Code examples and patterns
- `Migration-Guide.md` - Upgrading from legacy API
- `VS-Code-Extension.md` - Extension usage
